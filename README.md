## 🚀 開啟與部署方式

您可以直接使用瀏覽器開啟 `index.html`，或開啓 GitHub Pages 隨時查看！

### 使用 GitHub Pages 瀏覽 (推薦)
1. 前往倉庫設定 `Settings` -> `Pages`
2. Source 選擇 `Deploy from a branch`
3. Branch 選擇 `main` / `/ (root)` 點擊 `Save`
4. 即可獲得永久專屬網址！
